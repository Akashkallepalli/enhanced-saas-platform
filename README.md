# Enhanced SaaS Platform 
 
A scalable multi-tenant SaaS application built with **Node.js**, **Express**, **PostgreSQL**, and **React**. 
 
## ?? Features 
- **Multi-tenancy Architecture**: Secure data isolation for different organizations. 
- **Role-Based Access Control (RBAC)**: Granular permissions for Admins and Users. 
- **Authentication**: Secure JWT-based login and registration. 
- **Audit Logging**: Track all user activities for compliance. 
 
## ??? Tech Stack 
- **Backend**: Node.js, Express.js 
- **Database**: PostgreSQL (with custom migrations) 
- **Frontend**: React (Vite) 
- **DevOps**: Docker support included 
 
## ?? Installation 
1. Clone the repo 
2. Setup PostgreSQL database 
3. Run `npm install` 
4. Start the server with `npm start`
